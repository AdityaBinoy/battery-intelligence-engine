# Adaptive Multi-Cell Battery Intelligence Engine

## Overview

This project implements a battery intelligence engine capable of monitoring a simulated 4-cell lithium-ion battery pack in real time.

## Features

- Individual cell voltage monitoring
- Pack average voltage calculation
- Cell imbalance detection
- Strongest and weakest cell identification
- Battery health classification
- Real-time dashboard updates

## Health States

Healthy
Minor Imbalance
Critical Imbalance
Pack Failure

## Formulae

Average Voltage:

Vavg = (V1+V2+V3+V4)/4

Imbalance Percentage:

((Vmax-Vmin)/Vavg)×100
